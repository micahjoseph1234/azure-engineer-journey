# Lab 05 - Azure DNS

## Objective

Create a DNS Zone and configure DNS records.

## Resources Created

* DNS Zone: micahlab.com

## Steps Performed

### Step 1

Created DNS Zone named micahlab.com.

### Step 2

Created an A Record.

Name: www

Type: A

IP Address: 1.1.1.1

TTL: 3600

### Step 3

Verified record creation.

## Key Learnings

* DNS translates names into IP addresses.
* A Records map hostnames to IPv4 addresses.
* Azure DNS provides highly available DNS hosting.
